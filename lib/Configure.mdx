  <div className='sb-section-title'>
    <h1 className="gradient-text">🎨 BDS (Buzzle Design System)</h1>

      BDS는 Buzzle의 디자인 시스템으로, 사용자 경험의 일관성과 개발 효율성을 극대화하기 위해 철저히 계획되고 구현된 체계입니다. 이 시스템은 단순한 디자인 가이드라인을 넘어서, Buzzle의 브랜드 정체성과 사용자 중심 철학을 구현하는 핵심 도구입니다.


    <h2 className="section-title">🌟 핵심 철학과 원칙</h2>

    <div className="principle-grid">
      <div className="principle-card">
        ### 일관성의 힘
        모든 제품과 플랫폼에서 일관된 디자인 언어를 사용하여 예측 가능하고 신뢰할 수 있는 경험을 제공합니다.
      </div>
      <div className="principle-card">
        ### 포용적 접근성
        모든 사용자가 쉽게 이용할 수 있는 universal design의 철학을 반영합니다.
      </div>
      <div className="principle-card">
        ### 효율성의 극대화
        개발자와 디자이너 간의 원활한 협업을 촉진하고, 창의적인 문제 해결에 더 많은 시간을 할애할 수 있게 합니다.
      </div>
      <div className="principle-card">
        ### 유연한 확장성
        다양한 상황과 요구사항에 적응할 수 있는 컴포넌트와 패턴을 제공합니다.
      </div>
    </div>

      </div>

    <div className="design-token-grid">
      <div className="design-token-card">
        ### 🌈 색상 이론
        BDS의 색상 팔레트는 심리학적 영향, 접근성 기준, 그리고 브랜드 개성을 고려하여 신중하게 선택되었습니다. 각 색상은 특정한 감정과 메시지를 전달하도록 설계되었습니다:
        <ul>
          <li>🖼️ 배경 ({bds.color.background.base}): 안정감과 신뢰를 주는 톤으로, 콘텐츠를 돋보이게 합니다.</li>
          <li>🔘 버튼 ({bds.color.button.base}): 행동을 유도하는 생동감 있는 색상으로, 사용자의 주의를 끕니다.</li>
          <li>🖼️ 테두리 ({bds.color.border.base}): 요소 간의 구분을 명확히 하면서도 시각적 조화를 이룹니다.</li>
          <li>🔣 아이콘 ({bds.color.icon.base}): 직관적인 인식을 돕는 색상으로, 정보의 빠른 전달을 지원합니다.</li>
          <li>👥 그림자 ({bds.color.shadow.base}): 깊이감을 더해 UI의 계층 구조를 강화합니다.</li>
          <li>➖ 선 ({bds.color.line.base}): 미묘한 구분을 통해 레이아웃의 구조를 명확히 합니다.</li>
        </ul>
      </div>
      <div className="design-token-card">
        ### ✍️ 타이포그래피 철학
        BDS의 타이포그래피 시스템은 가독성, 계층 구조, 그리고 브랜드 개성을 균형 있게 반영합니다:
        <ul>
          <li>기본 텍스트 ({bds.typography.text.base.fontSize}px / {bds.typography.text.base.lineHeight}px): 읽기 쉽고 명확한 본문 텍스트를 제공합니다.</li>
          <li>로고 ({bds.typography.logo.base.fontSize}px / {bds.typography.logo.base.lineHeight}px): Buzzle의 독특한 브랜드 개성을 시각적으로 표현합니다.</li>
        </ul>
        제목 스타일은 정보의 계층 구조를 명확히 하고, 사용자의 주의를 효과적으로 가이드합니다:
        <ul>
          <li>📢 큰 제목 ({bds.typography.title.largeTitle.fontSize}px / {bds.typography.title.largeTitle.lineHeight}px): 주요 섹션을 강조하여 페이지의 구조를 명확히 합니다.</li>
          <li>🗣️ 기본 제목 ({bds.typography.title.base.fontSize}px / {bds.typography.title.base.lineHeight}px): 중요한 하위 섹션을 구분하여 정보의 흐름을 안내합니다.</li>
          <li>💬 작은 제목 ({bds.typography.title.small.fontSize}px / {bds.typography.title.small.lineHeight}px): 세부 정보를 효과적으로 분류하여 스캔 가능성을 높입니다.</li>
        </ul>
      </div>
      <div className="design-token-card">
        ### 📏 간격과 크기 이론
        BDS의 간격과 크기 시스템은 일관된 사용자 경험을 제공하고 시각적 리듬을 창출합니다:
        <ul>
          <li>🐜 작은 크기 : 세밀한 조정이 필요한 요소에 사용됩니다.</li>
          <li>🐝 중간 크기 : 대부분의 일반적인 UI 요소에 적용됩니다.</li>
          <li>🦋 큰 크기 : 강조가 필요한 요소나 넓은 공간에 사용됩니다.</li>
          <li>📏 기본 간격 : 요소 간의 기본적인 여백을 제공합니다.</li>
          <li>🔳 기본 테두리 두께 : 요소의 경계를 명확히 하면서도 과하지 않은 두께입니다.</li>
          <li>🔲 기본 테두리 반경 : 부드러운 모서리로 현대적인 느낌을 줍니다.</li>
        </ul>
      </div>

      </div>

      </div>

      </div>
    </div>
